How to calculate Product Area

-You can calculate it through the setting and check in units of measure.
Then you must check in variants and click on attributes.

Select radio size and write values like (0.50*0.20*12)

-(0.50*0.20 means area) (12 means number of product in the same box)

Enter new product and select the units of measure in meters

Then open attributes & variants and select values you need.

When you purchase or sales you must enter quantity by meters and press on calculate M to calculate number of units and boxes automatically.

If you want to calculate meters you should entre number of units and boxes and press on calculate box to calculate number of meters automatically.

Calculate Product Scrap

-If you want to calculate scrap you should select scrap through operations in inventory.

Then you must enter product, number of carton and number of tiles to calculate quantity by meters automatically.
